# Main Window Repositioning

**Goal**: Verify that the main application window can be dragged and repositioned by clicking and dragging the empty space in the header (between selectors).

1.  **Precondition**: 
    - App Main Window is visible.
    - `ServiceSelector` and `SessionSelector` are present.
2.  **Action**:
    - User clicks and drags from the free space between the Service Selector and Session Selector.
3.  **Expected Result**:
    - The window moves significantly from its original position.
4.  **Action**:
    - User drags the window again towards the top-center of the screen.
5.  **Expected Result**:
    - The window is repositioned near the top of the screen (Reset flow).
