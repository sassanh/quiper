# Auto-Download Updates

**Goal**: Verify app automatically downloads updates when enabled.

1.  **Precondition**: "Automatically download updates" is enabled.
2.  **Action**: New update is detected.
3.  **Expected Result**:
    -   Update package (.zip or .dmg) is downloaded in background.
    -   Status changes to "Ready to install".
    -   User is prompted to install/relaunch.
