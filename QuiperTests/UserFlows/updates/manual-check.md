# Manual Update Check

**Goal**: Verify user can manually check for updates.

1.  **Precondition**: Settings window is open, "Updates" section is visible.
2.  **Action**: User clicks **Check for Updates**.
3.  **Expected Result**:
    -   Status changes to "Checking...".
    -   If update available: Shows version and release notes.
    -   If up to date: Shows "You're up to date".
