# Reordering Services

**Goal**: Verify user can reorder services via drag and drop.

1.  **Precondition**: Multiple services configured.
2.  **Action**: User drags a service icon in the bottom bar to a new position.
3.  **Expected Result**:
    -   Service icons reorder visually.
    -   New order is persisted in Settings.
    -   Service switching shortcuts respect the new order.
