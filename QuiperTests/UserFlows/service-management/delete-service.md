# Deleting a Service

**Goal**: Verify user can delete a service.

1.  **Precondition**: Settings window open, service selected.
2.  **Action**: User clicks "Delete Service" (or context menu delete).
3.  **Action**: User confirms deletion in dialog.
4.  **Expected Result**:
    -   Service is removed from list.
    -   Associated custom action scripts are deleted.
    -   If active, overlay switches to another service.
