# Find in Page

**Goal**: Verify user can search text within the active WebView.

1.  **Precondition**: Overlay is visible.
2.  **Action**: User triggers Find shortcut (`Cmd + F`).
3.  **Expected Result**:
    -   Find bar appears.
    -   Input field is focused.
4.  **Action**: User types text.
5.  **Expected Result**:
    -   Matches are highlighted in WebView.
    -   Match count is displayed.
